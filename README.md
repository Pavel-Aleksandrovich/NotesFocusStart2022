# Notes
Application for watching list of notes or tasks

## Branch
- Current branch - ** main **

## Features
- List notes
- Create note
- Save note
- Changing exists note
- Different layouts for portrait and landscape

## Stack
- Swift
- UIKit
- VIPER
- UserDefaults
- UI programmatically (Constraints). Without Storyboard
- DI
